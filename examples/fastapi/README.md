# FastAPI on Azure Functions via Web Adapter

A standard FastAPI application running on Azure Functions **with zero code changes**.

## Project Structure

```
fastapi/
├── host.json
├── worker.config.json
├── Dockerfile
└── app/
    ├── main.py               # Standard FastAPI app (NO CHANGES)
    └── requirements.txt
```

## Configuration

| Variable | Value | Purpose |
|---|---|---|
| `AZURE_FWA_PORT` | `8080` | Port Uvicorn listens on |
| `AZURE_FWA_STARTUP_COMMAND` | `uvicorn main:app --host 0.0.0.0 --port 8080` | Start with Uvicorn |
| `AZURE_FWA_READINESS_CHECK_PATH` | `/` | Health check path |

## Run Locally

```bash
cd app
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8080
# → http://localhost:8080
# → http://localhost:8080/docs  (Swagger UI)
```

## API Endpoints

| Method | Path | Description |
|---|---|---|
| GET | `/` | Welcome message |
| GET | `/api/hello?name=X` | Hello with name parameter |
| POST | `/api/echo` | Echo back JSON body |
| GET | `/api/health` | Health check |
| GET | `/api/items/{id}` | Get item by ID |
| GET | `/docs` | Swagger UI (auto-generated by FastAPI) |
